## Build and Deploy a Google Search Clone using React & Tailwind CSS
##### from https://youtu.be/NDbruK1fzG8
##### by JavaScript Mastery

### Steps
1. ```npx create-react-app ./ (指定裝在跟目錄下)```
2. ```移除/src 所有檔案```
3. ```建立 index.js 和 app.js (可以裝Extension)```
4. ```npm install -D tailwindcss@npm:@tailwindcss/postcss7-compat postcss@^7 autoprefixer@^9```
5. ```npm install @craco/craco```
6. ```修改package.json 的 scripts模式```
7. ```新增craco.config.js```
8. ```npx tailwindcss-cli@latest init```
9. ```修改 tailwind.config.js 的 purge```
10. ```新增global.css 並在 index.js 引用```
(https://tailwindcss.com/docs/guides/create-react-app#setting-up-tailwind-css)
